Windows Programs That Make It Usable

File Converter

Yes, that's what it's called.

It adds file conversions to the Explorer context menu, and it's very useful.

Unfortunately, it's been kind of ruined, so I will be distributing an installer of an older version.

10/10 Program


QuickLook

This program essentially brings macOS file previews to Windows.

Yes, PowerToys also has this feature, but I like this program better.

8/10


PowerToys

This program is a must-have for Windows.

Things I use the most from it include:

  • The Color Picker
  • The Text Selector
  • PowerToys Run
  • FancyZones
  • Shortcut Remapper (I remap Alt+F4 to Ctrl+Q)

11/10


mp3tag

I have a big music collection, and this makes editing their properties much easier.

It is far better than iTunes or Windows Media Player (which doesn't let you edit FLAC files).

9/10


Honorable Mentions

  • 7-Zip Best folder archiver
  • Rufus / Balena Etcher Great for creating USB drive installers
  • LocalSend Transferring files between devices
  • YouTube Downloader A really good YouTube video downloader
  • Plex Movies and such
  • FlairMax Basically Shazam for Windows
  • New Super Ultimate Injector Great for custom virtual console ROMs on 3DS
  • AcrylicMenus Makes the "More Options" context menu have acrylic

Other Things I Do

  • Make the default File Explorer context menu the "More Options" one
  • Not use the default cursor (I'm gatekeeping the cursor set I use)
  • Turn desktop icons off
  • Turn off as many ads as possible
  • Probably a lot of other things I forgot about
